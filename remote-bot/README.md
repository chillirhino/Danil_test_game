# 🦫 Remote bot — управляй Claude Code с телефона через Telegram

Пишешь боту в Telegram → он запускает Claude Code в этом проекте на твоём ПК → присылает ответ обратно. Доступ строго по твоему `chat id`.

## Что нужно
- ПК включён, есть интернет, установлен **Node.js 18+** и **Claude Code** (`claude` в PATH).
- Приложение **Telegram** на телефоне.

## Настройка (один раз, ~3 минуты)

1. **Создай бота.** В Telegram напиши **@BotFather** → `/newbot` → задай имя. Он пришлёт **токен** вида `123456:ABC-...`.

2. **Создай config.** Скопируй `config.example.json` в `config.json` и вставь токен:
   ```json
   {
     "botToken": "СЮДА_ТОКЕН_ОТ_BOTFATHER",
     "allowedChatId": "GET_MY_ID",
     "projectDir": "D:\\Unity Projects\\Danil_test_game",
     "skipPermissions": true,
     "claudeBin": "claude",
     "timeoutMinutes": 15
   }
   ```

3. **Узнай свой chat id.** Запусти бота (двойной клик по `start-bot.bat`, или `node bot.js`). Напиши своему боту в Telegram любое сообщение — он ответит `Твой chat id: 12345678`.

4. **Впиши id.** Останови бота (закрой окно), в `config.json` замени `"GET_MY_ID"` на свой номер, например `"allowedChatId": "12345678"`. Запусти бота снова.

5. **Готово.** Теперь пиши боту с телефона откуда угодно. Пример: «сделай в Level 2 первый провал уже» — Claude выполнит и ответит.

## Команды в чате
- `/new` — начать новый разговор (сбросить контекст).
- `/id` — показать твой chat id.
- `/help` — справка.

Контекст разговора сохраняется между сообщениями (через session id Claude Code), пока не нажмёшь `/new` или не перезапустишь бота.

## Чтобы работало, пока тебя нет
- **ПК не должен засыпать.** Параметры электропитания → «Сон» → Никогда (хотя бы при питании от сети).
- Оставь окно `start-bot.bat` открытым (оно само перезапустит бота, если тот упадёт).
- Хочешь автозапуск при включении ПК: `Win+R` → `shell:startup` → положи туда ярлык на `start-bot.bat`.
- Unity держи открытым, если хочешь править игру удалённо (Claude общается с редактором Unity локально).

## ⚠️ Безопасность — прочитай
- Этот бот = **удалённое управление ПК**: Claude может запускать команды и менять файлы (при `skipPermissions: true` — без подтверждений).
- Доступ разрешён **только твоему chat id**. Любой другой получает «Not authorized».
- **Держи `botToken` в секрете.** Кто угодно с токеном может писать боту (но выполнять сможет только с разрешённого chat id). `config.json` уже в `.gitignore` — не попадёт в git.
- Если хочешь безопаснее, но с подтверждениями — поставь `"skipPermissions": false`. Тогда команды, требующие разрешения, зависнут (бот не сможет ответить на диалог), поэтому для удалённой работы обычно оставляют `true`. Решай сам.
- Каждое сообщение = отдельный запуск Claude Code (тратит токены/лимит твоего аккаунта).

## Если не работает
- «Не удалось запустить claude» → укажи полный путь в `claudeBin`, например `"claudeBin": "C:\\Users\\ТЫ\\AppData\\Roaming\\npm\\claude.cmd"`.
- «Bad bot token» → проверь токен от BotFather.
- Бот не отвечает → проверь, что окно `start-bot.bat` открыто и есть интернет.
